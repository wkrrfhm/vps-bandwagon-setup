## 拿到 VPS 之后：KiwiVM 控制面板操作

KiwiVM 是搬瓦工自研的控制面板，用起来比一般面板直观。登录方式：进客户端 → 找到购买的 VPS → 点击旁边的 Open KiwiVM。

几个常用功能：

- **Main Controls**：显示 IP 地址、SSH 端口，开关机、重启都在这里
- **Root Password Modification**：重置 root 密码（要先关机才能操作，重置完密码只出现一次，必须马上复制保存）
- **Reinstall OS**：重装系统，20 多个模板，推荐 Debian 12 或 Ubuntu 22.04
- **Snapshot**：创建快照，装好基础环境后拍一个快照，以后搞坏了直接回滚

说个容易踩的坑：**重置密码前一定要先把 VPS 关机**，开机状态下点重置密码会失败，而且不给任何提示。

## SSH 连接：Windows 和 Mac 分别怎么做

### Windows 用户

推荐用 Termius，免费版够用，界面直观，配置可以云端同步。也可以用 Xshell 或者 PowerShell 直接命令行连。

PowerShell 连接方式（Windows 10 以上自带）：

ssh root@你的IP地址 -p 你的SSH端口号

比如：`ssh root@123.456.78.90 -p 28866`

回车之后会提示输入密码，输密码时屏幕上不显示任何字符，直接盲打然后回车就行。

### Mac 用户

Mac 自带终端就够，打开 Terminal，输入：

ssh root@你的IP地址 -p 你的SSH端口号

和 Windows PowerShell 完全一样的命令。

登录成功后会看到 Linux 的命令行提示符，通常是 `root@localhost:~#` 这种样子。

## 登录之后能干什么：几个常见搭建场景

### 场景一：WordPress 建站

搬瓦工上跑 WordPress 的人挺多的，配合 LNMP 一键安装包最省事。

1. **登录 SSH 后，更新系统**：
   
   apt update && apt upgrade -y
   

2. **安装 LNMP**（Nginx + MySQL + PHP 组合）：
   去 LNMP 官方下载安装脚本，按提示选择版本，整个过程大概 20-40 分钟，期间不需要操作

3. **添加虚拟主机**：
   
   lnmp vhost add
   
   按提示输入域名，LNMP 会自动配置 Nginx

4. **安装 WordPress**：下载 WordPress 压缩包，上传到对应目录，浏览器访问域名完成图形化安装

顺便提一嘴，WordPress 站不装缓存插件的话 TTFB 普遍在 800ms 以上，装上缓存之后能压到 200ms 左右，这个差距肉眼能感觉到。

### 场景二：部署 Docker 应用（比如 n8n）

1. **安装 Docker**：
   
   curl -fsSL https://get.docker.com | sh
   

2. **拉取并运行 n8n**：
   
   docker run -d --name n8n -p 5678:5678 -v ~/.n8n:/home/node/.n8n n8nio/n8n
   

3. **浏览器访问** `http://你的IP:5678`，完成 n8n 初始设置

CN2 GIA-E 套餐的 1GB 内存跑 n8n 够用，如果同时还要跑数据库就有点紧，建议上 2GB 内存的套餐。

### 场景三：纯 Linux 学习环境

如果只是学 Linux 操作，最轻量的用法就是登录 SSH 然后开始练命令。推荐先过一遍这几个：文件系统（ls、cd、cp、mv、rm）、权限管理（chmod、chown）、进程管理（ps、kill、top）、网络（netstat、curl、wget）。

## 常见问题 FAQ

**Q：搬瓦工 VPS 搭建教程里，新手第一个套餐选哪个？**

**Q：VPS IP 被封了怎么办？**

搬瓦工换 IP 收 $8 一次，在 KiwiVM 里操作。CN2 GIA-E 套餐的优势是可以迁机房，换个机房 IP 就变了，比单独换 IP 灵活。

**Q：搬瓦工支持中文吗？支付宝可以付款吗？**

官网是英文界面，但支付支持支付宝，购买流程里可以直接用支付宝扫码付款。账户语言切不了中文，但对照着截图操作没什么难度。

**Q：SSH 连不上怎么排查？**

按顺序查：1）VPS 是否在运行状态（KiwiVM 里确认）；2）IP 和端口号是否填对（端口一般是五位数，不是默认的 22）；3）密码有没有重置过（初始密码只在开机邮件里出现一次，如果没保存就去 KiwiVM 重置）；4）本地网络是否正常。

**Q：搬瓦工 VPS 能跑多少个网站？**

取决于内存和流量。1GB 内存跑 3-5 个小流量 WordPress 站问题不大，但装了 MySQL 之后内存会占掉一大块，如果要跑较多站或者数据库，考虑更高配置的套餐。
